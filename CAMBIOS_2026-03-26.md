### 1. Error de Inicio - `NameError: name 'Index' is not defined`

**Problema:** La aplicación no iniciaba, mostrando error en los logs:
```
Error al inicializar base de datos: name 'Index' is not defined
```

**Causa:** Varios archivos de modelos usaban `Index` sin importarlo.

**Solución:** Agregado `Index` a los imports en:

| Archivo | Estado | Cambio |
|---------|--------|--------|
| `models/cajero.py` | ❌ → ✅ | Agregado `Index` al import |
| `models/cotizacion.py` | ❌ → ✅ | Agregado `Index` al import |
| `models/proveedor.py` | ❌ → ✅ | Agregado `Index` al import |
| `models/cliente.py` | ✅ | Ya tenía |
| `models/producto.py` | ✅ | Ya tenía |

**Código corregido (ejemplo):**
```python
# ANTES (cajero.py)
from sqlalchemy import Column, Integer, String, Boolean

# DESPUÉS
```

### 2. Gráficos no Disponibles - `matplotlib no está instalado`

**Problema:** El dashboard mostraba:
```
📊 Gráficos no disponibles
matplotlib no está instalado
```

**Causa:** Conflictos de compatibilidad entre matplotlib y PySide6 en el EXE compilado.

**Solución:** Reemplazo completo de `chart_widget.py` para usar **gráficos nativos de Qt** sin matplotlib.

**Ventajas:**
- ✅ Más rápido y ligero
- ✅ Sin dependencias externas problemáticas
- ✅ 100% compatible con PySide6
- ✅ Gráficos vectoriales escalables
- ✅ EXE más pequeño (~15MB menos)

**Archivo modificado:** `tucajero/ui/chart_widget.py`

**Nueva implementación:**
```python
# Usa QPainter para dibujar gráficos nativamente
from PySide6.QtGui import QPainter, QColor, QPen, QBrush

class ChartWidget(QWidget):
    def paintEvent(self, event):
        painter = QPainter(self)
        painter.setRenderHint(QPainter.RenderHint.Antialiasing)
        # Dibuja barras o torta según corresponda
```

**Actualización del spec:**
```python
# TuCajero.spec - Eliminadas dependencias innecesarias
```

### 5. Persistencia de Carrito

**Archivo:** `ui/ventas_view.py`

**Problema:** Si la app se cerraba bruscamente, se perdía el carrito.

**Solución:** Guardado automático en archivo JSON.

```python
# Al iniciar
self.cart_file = Path(get_data_dir()) / "temp" / "carrito_temporal.json"
self.load_cart()  # Recuperar si existe

# Después de cada modificación
def actualizar_tabla(self):
    # ... actualizar UI ...
    self.save_cart()  # Guardar automáticamente

# Después de venta exitosa
def _confirmar_cobro(self):
    # ... registrar venta ...
    self.clear_cart_file()  # Limpiar archivo
```

**Formato del archivo:**
```json
{
  "items": [
    {
      "producto_id": 5,
      "cantidad": 2,
      "precio": 5000,
      "aplica_iva": true
    }
  ],
  "descuento": {"tipo": "porcentaje", "valor": 10, "total": 1000},
  "cliente_id": null
}
```

### 6. Índices de Base de Datos

**Archivo:** `config/database.py`

**Función agregada:** `crear_indices()`

**14 índices creados:**

| Índice | Tabla | Columna | Propósito |
|--------|-------|---------|-----------|
| `idx_productos_codigo` | productos | codigo | Búsqueda por código de barras |
| `idx_productos_activo` | productos | activo | Filtrar activos |
| `idx_productos_nombre` | productos | nombre | Búsqueda por nombre |
| `idx_ventas_fecha` | ventas | fecha | Reportes por fecha |
| `idx_ventas_anulada` | ventas | anulada | Filtrar no anuladas |
| `idx_ventas_cliente` | ventas | cliente_id | Historial cliente |
| `idx_ventas_cajero` | ventas | cajero_id | Auditoría |
| `idx_clientes_nombre` | clientes | nombre | Búsqueda cliente |
| `idx_clientes_activo` | clientes | activo | Filtrar activos |
| `idx_movimientos_producto` | movimientos_inventario | producto_id | Movimientos por producto |
| `idx_movimientos_fecha` | movimientos_inventario | fecha | Reportes |
| `idx_movimientos_producto_fecha` | movimientos_inventario | producto_id, fecha | Índice compuesto |
| `idx_cortes_fecha` | cortes_caja | fecha_apertura | Cortes por fecha |

**Código:**
```python
def crear_indices():
    indices = [
        "CREATE INDEX IF NOT EXISTS idx_productos_codigo ON productos(codigo)",
        "CREATE INDEX IF NOT EXISTS idx_ventas_fecha ON ventas(fecha)",
        # ... más índices
    ]
    
    with engine.connect() as conn:
        for idx_sql in indices:
            conn.execute(text(idx_sql))
        conn.commit()
```
