## 🎓 FILOSOFÍA DEL DISEÑO

**Psicología del color premium:**
- Ultra oscuro (#0f1320): Reduce fatiga visual, sensación premium, focus en contenido
- Gradientes vibrantes: Jerarquía visual, modernidad, atención en métricas clave
- Texto blanco puro: Máximo contraste, legibilidad profesional
- Bordes sutiles: Separación sin estridencia
- Espaciado uniforme: Organización, profesionalismo, reduce carga cognitiva

**Tipografía:**
- H1 (36px): Títulos de página
- H2 (28px): Secciones principales  
- H3 (22px): Subsecciones
- Body (14px): Texto normal
- Caption (12px): Ayudas y labels

**Espaciado (base 8px):**
- 8px: Entre elementos relacionados
- 16px: Entre secciones
- 24px: Entre cards/contenedores
- 32px: Márgenes de página
